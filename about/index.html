The Quiz

There's a lot of trivia about the Trek franchise scattered throughout the web. I'm hoping trektrivia.com can become a place where that trivia (and more) can centralize, so Trekkies can flock here to get their dose of trivia and flex their muscles. The site is built to accommodate a vast number of questions and relies on constant user submission to keep the questions fresh.

Writing a Question

I've tried my best to phrase many questions in this format:

Captain Picard prefers this type of tea.

Where "this" represents the correct answer. This seems like an eloquent and easy way to frame a question.

The hardest part is brainstorming plausible alternative responses. When you're writing a question, try to pick a topic that seems conducive to alternatives.

All information should be verifiable on Memory Alpha (i.e. questions should be canon).

About the Creator

I've been a Trek fan since primary school, when my parents introduced me to The Next Generation. DS9 shortly followed and became my favorite series. The optimistic and principled attitudes of many characters in the franchise are powerful; this optimism is likely the reason why I adore the series so much.

Making this quiz is my first venture into programming, so it's neither efficient nor flawless, and I'd welcome feedback. If you have any suggestions about the code and/or structure of the quiz, don't hesitate to message me using the form linked above.

Notes...

  • I tend to misspell things... if you find an error, submit it here.
  • Questions about TOS, ENT, DIS, and PIC are in short supply! If you know these series well, please submit some questions about them!
  • If you've got a suggestion for a different font (perhaps one with a more futurisic vibe), send me an message using the link above!
  • This quiz relies heavily on the added functionality of EMCAScript6 (over the prior ES5). If the game isn't running, your browser's support for ES6 is likely the culprit.
